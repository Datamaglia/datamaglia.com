Get Started wth Datamaglia Recommendations

This tutorial will show how Datamaglia Recommendations can be used to enhance user engagement in a photo browsing app. We will show how to request a Datamaglia API key, how to configure your Datamaglia application and how to use the Datamaglia Recommendations API to generate personalized recommendations and show these to the user.

Introducing Picmaglia

This tutorial will assume we are a developer with a basic photo browsing app and how we can add personalized recommendations.

Overview

These are the basic features that we've already implemented for Picmaglia. Users are able to

  • Browse photos: view the most recently shared photos.
  • Favorite photos: select photos they like and save these for viewing later.
  • View favorited photos: see all the photos that user has favorited.
  • View recommended photos What photos is this user likely to be interested in?

These features are great, but we'd like to increase user engagement with our app by adding a way for users to discover new photos (other than by viewing recent photos). We'd like to be able to suggest new photos to users based on the photos they have previously favorited.
